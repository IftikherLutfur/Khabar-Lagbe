# 🍽 HungryNaki – Restaurant Website

HungryNaki is a modern restaurant web application where customers can browse the menu, filter food items, place orders, and even reserve tables online. It features an **Admin Dashboard** for managing the platform efficiently.

---

## ✨ Features

### 🍴 User Features
- **Menu Browsing** – View delicious food items.
- **Food Filtering** – Filter menu items by:
  - Breakfast
  - Lunch
  - Dinner
- **Sorting** – Sort food items based on price.
- **Order Food** – Add items to the cart and place an order.
- **Secure Payment** – Pay directly from the cart.
- **Table Reservation** – Reserve your table in advance.

### 🛠 Admin Features
- **Admin Dashboard**
- **Admin Role Access Control**
- Manage menu items, orders, and reservations.

---

**Backend:**
